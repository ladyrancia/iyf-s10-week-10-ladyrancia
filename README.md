## Project Description
This project is a Node.js and Express REST API built for Week 10 backend development. The API handles posts, users, and comments using structured routes and controllers.

---

## Technologies Used
- Node.js
- Express.js
- TypeScript
- JavaScript
- npm
- REST API

---

## Features
- Get all posts
- Get a single post
- Create posts
- Update posts
- Delete posts
- API routing with Express
- JSON responses

---

## How to Run

## Clone the repository

```bash
git clone https://github.com/ladyrancia/iyf-s10-week-10-ladyrancia.git

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

## Open in Browser

```txt
http://localhost:3000/api/posts
```
